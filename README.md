# DSA LeetCode Daily Challenge 🚀

Welcome to my **Daily LeetCode Challenge** repository!  
Every day, I solve a LeetCode problem to improve my **Data Structures & Algorithms (DSA)** skills.

## Structure

- **Problems/** – Contains daily problems organized by day.  
- **Notes/patterns.md** – Notes on algorithmic patterns, tricks, and techniques.  
- **Utils/helper_functions.py** – Helper functions used across multiple solutions.

## How to follow along

1. Browse the **Problems** folder.  
2. Check the daily problem solution.  
3. Learn from the explanations and comments in the code.  
4. Feel free to **fork** and try the problems yourself!
